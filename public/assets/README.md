# Hướng dẫn thay thế hình ảnh (Asset Replacement Guide)

Dưới đây là danh sách các file ảnh cần thiết cho Landing Page Thang Máy Tín An. Bạn vui lòng thay thế các file placeholder bằng ảnh thật của dự án.

## 1. Màn hình chính (Hero Section)
- `hero-bg.jpg`: Ảnh nền màn hình đầu tiên. Nên chọn ảnh thang máy đẹp nhất, có độ phân giải cao và được làm mờ/tối nhẹ để nổi bật chữ.

## 2. Giá trị cốt lõi (Value Proposition)
- `icon-class.png`: Icon biểu tượng "Đẳng cấp".
- `icon-filial.png`: Icon biểu tượng "Hiếu nghĩa".
- `icon-tailor.png`: Icon biểu tượng "May đo công nghệ".

## 3. Câu chuyện Founder
